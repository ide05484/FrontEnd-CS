## 인터럽트 개념
#### 1. CPU에게 이벤트 발생을 알리는 신호
- 외부/내부 이벤트로 실행 중인 작업을 잠시 중단하고 처리

#### 2. 종류
- 외부 인터럽트 : I/O 완료, 타이머, 키보드 입력
- 내부 인터럽트 : 예외, 시스템 콜

---

## 인터럽트 처리 과정

1. 인터럽트 발생
2. 현재 실행 중인 Context 저장
3. 인터럽트 핸들러 실행
4. 처리 완료 후 Context 복구
5. 원래 작업으로 복귀
